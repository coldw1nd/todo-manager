# React Todo Manager

### Ссылки
- **[GitHub Pages](https://coldw1nd.github.io/todo-manager/)**

### Функционал
- Добавление, удаление и отметка выполнения задач.
- Фильтрация (Все / Активные / Выполненные).
- Счетчик оставшихся задач.
- Сохранение данных в **localStorage** (задачи не пропадают после перезагрузки страницы).

### Дополнительно реализовано:
1. **Редактирование задачи:** Двойной клик по тексту задачи открывает поле ввода для изменения.
2. **Смена темы:** Переключатель меgжду темной и светлой темой оформления.

### 💻 Технологии
- React.js
- Vite
- CSS-in-JS
